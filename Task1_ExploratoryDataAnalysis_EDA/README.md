# Exploratory Data Analysis (EDA) - Titanic Dataset

## Objective
Perform exploratory data analysis on the Titanic dataset to identify trends and patterns affecting passenger survival.

## Tools & Libraries
- Python
- Pandas
- NumPy

## Analysis Performed
- Data Structure Analysis
- Descriptive Statistics
- Survival Rate Analysis
- Gender-wise Survival Analysis
- Passenger Class Analysis
- Missing Value Detection
- Hypothesis Testing

## Dataset
Titanic Dataset from GitHub.
